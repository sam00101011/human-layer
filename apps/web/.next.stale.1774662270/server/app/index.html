Human Layer web app

This app provides the Phase 0 API surface, a stubbed dev login flow, and the extension auth handoff used by the browser extension.

  • Page lookup API: /api/pages/lookup?url=...
  • Dev login: /dev-login
  • Extension handoff: /auth/extension-handoff

Suggested first run

  1. Start Postgres and Redis with docker compose.
  2. Run the database migration and seed scripts.
  3. Start the web app and extension shell.
  4. Open GitHub or a seeded Hacker News page to exercise the overlay states.
