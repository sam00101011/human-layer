# Product Thesis

## Main framing

Human Layer is not a new social network first.

It is humanhood middleware for the web:
- a verified-human comment layer
- a verified-human search layer
- a verified-human ranking and moderation layer
- a verified-human access control layer

## Why it matters

Most internet systems are shaped by:
- bots
- sockpuppets
- fake engagement
- spam comments
- noisy rankings
- low-trust moderation

Human Layer changes the rules from:
- one wallet, one account
- one cookie, one session
- one email, one identity

to:
- one human, one voice
- one human, one vote
- one human, one claim
- one human, one moderation seat

## Best initial wedge

Start with a verified-human annotation layer for high-value pages:
- GitHub
- docs sites
- Hacker News links
- YouTube
- arXiv
- major blogs

## Product shape

The strongest initial product is:
- browser extension
- web app
- optional iOS app later

The browser extension injects the overlay.
The web app powers profiles, search, saved pages, reputation, and moderation.
