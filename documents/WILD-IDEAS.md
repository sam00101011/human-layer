# Wild Ideas

## Human comment overlay

Add a verified-human comment layer on top of websites.

Not just a comment box:
- human summary
- page verdict
- vote counts from unique humans
- scam or outdated warnings
- one-human moderation

## Human search

Search ranked by verified-human interactions instead of bot engagement.

This could become:
- search over human-annotated pages
- search results ranked by real human usefulness
- a "best explanation" engine for the web

## Human Mode

A mode for browsing where:
- ads are hidden
- native comments are hidden
- bot noise is suppressed
- only verified-human commentary is shown

## Human checkpointing for AI

Require a verified-human approval before:
- payouts
- escalations
- publishing
- autonomous high-risk actions

## Humanhood middleware

The broadest frame is:
- not a single website
- not a single social app
- not even just a browser extension

But a general layer that adds verified humanness to:
- websites
- APIs
- search
- chat
- social
- moderation
- marketplaces
