# Platforms And Distribution

## Browser extension

This is the best first platform.

Why:
- works on the real web
- can inject a visible overlay
- can attach to existing pages
- ideal for comment overlay and page annotations
- best fit for Human Mode

## Web app

Use the web app for:
- search
- profiles
- moderation
- saved pages
- settings
- public result pages

The extension should feel like the capture and reading layer.
The web app should feel like the home base.

## iOS app

The best iOS MVP is:
- search
- profiles
- saved pages
- in-app browser with overlay
- share extension like "Open in Human Layer"

Do not treat iOS as a system-wide overlay platform at first.

## Full browser or Electrobun shell

A Chromium-based app could become a full Human Mode browser.

Benefits:
- stronger control over page rendering
- ad blocking built in
- consistent overlay behavior
- native ability to suppress site comments
- custom search and history integration

Costs:
- much bigger scope
- browser maintenance burden
- cookies, sessions, login, and security become your problem

Best path:
- extension first
- full browser later if the wedge is clearly strong

## Bluesky
